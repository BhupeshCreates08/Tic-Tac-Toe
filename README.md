# Tic Tac Toe 🎮
 
A simple and clean Tic Tac Toe game built with HTML, CSS, and JavaScript. Two players take turns, win conditions are checked automatically, and the board resets with one click.
 
 
## ✨ Features
 
- Two player game (O and X)
- Automatic winner detection
- Winner message display
- Reset & New Game button
- Hover animations on boxes and buttons
- Fully responsive design using `vmin` units
---
 
## 🛠️ Built With
 
- HTML5
- CSS3
- JavaScript (Vanilla)

## 🎮 How to Play
 
1. Player **O** always goes first
2. Players take turns clicking on empty boxes
3. First player to get 3 in a row (horizontally, vertically, or diagonally) wins
4. Click **Reset Game** or **New Game** to start over
